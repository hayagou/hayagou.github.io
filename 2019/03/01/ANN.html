ANN 인공신경망

Posted by hayagou on 2019-03-01 · 3 mins read

[인공지능] ANN(Artifical Neural Network)


배경

생리학자인 워렌 매컬록 (Warren McCulloch)과 수학자 월터 피트 (Walter Pitts)에 의해 1943년에 처음 소개 생물학적 뉴런이 동물 두뇌에서 어떻게 작동해서 명제 논리를 사용한 복잡한 계산을 수행하는 지를 모델링하기 위해 제안


인공 뉴런 (Artificial Neuron)

  • 생물학적 뉴런의 기능을 단순화해서 소프트웨어로 구현한 것
  • n개의 입력을 받고, 각각의 입력에 가중치를 곱한 후 이의 합을 구하고, 이 합에 간단한 함수를 적용하여 값을 출력
  • 결과 값이 다시 다른 뉴런의 입력으로 전달


퍼셉트론 (Perceptron)

  • 가장 단순한 ANN 아키텍처 중 하나로 Frank Rosenblatt가 1957년에 발명
  • 워렌 매컬록과 월터 피트가 제안한 모델과는 약간 다른 인공 뉴런을 사용


위 그림은 두 개의 입력을 받아 3 개의 출력을 내는 퍼셉트론 구조로,
입력 뉴런은 특수한 뉴런으로 입력 값을 그대로 통과시키는 역할
  • 퍼셉트론은 그림과 같이 입력 층과 여러 개의 LTU(Linear Threshold Unit)로 구성된 단일 층으로 구성
  • 출력은 아래와 같이 바이어스를 고려하여 입력들의 가중치 합이 일정 임계치 이상이며 1이고, 그렇지 않으면 0


  • 아래와 같이 재구성

임계값을 입력 값이 항상 1이고, 가중치가 -Θ인 입력 뉴런으로 표현
이렇게 되면 학습 데이터에 내재된 바이어스를 자동으로 학습


딥러닝 학습이란?

주어진 입력에 대해 원하는 출력 (목표치)이 나오도록 하는 가중치를 자동으로 찾는 작업 또는 과정

  • 아래의 학습 규칙에 따라 가중치 조절

  • 실제 출력과 목표치와의 차이, 즉 오차에 입력 값을 곱한 값만큼 변경
  • 오차가 음수이면 출력 값을 현재보다 작게 하기 위해 가중치를 감소
  • 반대로 오차가 양수이면 출력 값을 현재보다 크게 하기 위해 가중치를 증가


학습율, 학습 속도 제어

학습율이 너무 낮으면 알고리즘이 최적의 솔루션 (지역 최적의 솔루션 포함)에 도달하지만 시간이 오래 걸림


반대로 너무 높으면 알고리즘이 발산하여 최적의 솔루션을 찾을 수가 없음


-> 학습율에 대한 한 가지 해결책은 초반에는 학습율을 크게해서 학습 속도가 빠르게 진행되게 하고, 점차 학습율을 줄여 끝으로 갈수록 천천히 변하도록


학습 모드와 테스트 모드

  • 신경망은 두 가지 모드로 동작하는데, 하나는 학습 모드이고, 다른 하나는 사용 모드 (또는 테스트 모드)
  • 보통 데이터의 80%를 훈련에 사용하고 20%는 테스트용으로 사용
